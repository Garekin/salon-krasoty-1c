# Салон красоты — учётная система на 1С:Предприятие 8.3

Дипломный проект (Яндекс Практикум, профессия «1С-разработчик»). Комплексная
конфигурация автоматизации салона красоты, спроектированная и разработанная с
нуля, вместе с сопряжённым мобильным приложением на платформе 1С.

Исходники выгружены через `Выгрузить конфигурацию в файлы` (DumpConfigToFiles) —
формат, пригодный для версионирования в Git.

> **Автор:** Маркарян Гарекин · **Версия конфигурации:** 2.0.0.1

---

## Мобильное приложение (`Mobile/`)

Отдельная конфигурация под мобильную платформу 1С с двусторонней синхронизацией
данных с основной системой через план обмена.

---

## Реализованная функциональность

- **Складской учёт** — партионный учёт по методу **FEFO** (первым списывается
  партия с ближайшим сроком годности).
- **Регламентированный учёт** — план счетов, регистр бухгалтерии, формирование
  ОСВ и проводок (Дт 44 / Кт 70).
- **Интеграции** — обмен данными по **REST API** (HTTP-сервисы, парсинг JSON),
  умная загрузка прайс-листов из `.xlsx`.
- **Мобильное приложение** — на платформе 1С с двусторонней синхронизацией.
- **Разграничение прав доступа** — 27 ролей.
- **Аналитика** — дашборды и отчёты на **СКД**.

## Как развернуть

Конфигурация загружается из исходников в пустую информационную базу:

**Через конфигуратор:**
`Конфигурация` → `Загрузить конфигурацию из файлов…` → указать папку `Configuration/`
(для мобильной — `Mobile/`).

**Через командную строку:**
```
1cv8 DESIGNER /F "путь_к_базе" /LoadConfigFromFiles "путь_к_репозиторию/Configuration"
```

После загрузки выполнить `Обновить конфигурацию базы данных` (F7).
